## 2. Vấn đề dữ liệu

2.2. Vấn đề liên quan chọn file Markdown hay Text. 
2.3. Cân nhắc việc đa dạng câu hỏi ( crawl câu hỏi từ các trang xã hội của khoa trường)
* Markdown có thể giữ được định dạng bảng và xử lý được các thông tin dạng bảng nhưng vấn đề ở việc dữ liệu có thể không được "sạch" gây ra nhiễu thông tin, cần phải xử lí thêm vào file markdown.
* Text cải thiện được các nhược điểm trên nhưng sẽ bị mất dữ liệu bảng, cần phải cung cấp thêm.
